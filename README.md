# react-admin-plus

## 简介

基于react、ant-ui、typescript的前端微服务框架。欢迎star。

## 在线地址

<a href="https://www.liuguisheng.vip/react-admin-plus/#/login" target="_blank">在线demo</a>

## 项目特色

- 全面适配移动端
- 集成外链
- 多级菜单，常用组件，地图，富文本，echarts等
- 页面可配置
- 主页更加灵活，常用页面快速访问
- 全局主题配置
- 接口，业务代码，组件分离，代码更清晰，适合大型项目
- 集成animation库，内置丰富动画
